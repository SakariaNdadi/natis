Welcome to the Learners License Mock Exam Simulator! This project is an open-source initiative inspired by Frans Nekongo's Namibia Learners Licence. It is tailored to help aspiring drivers evaluate their competency before taking the official learners license examination. By providing a simulated exam experience, this tool aims to boost confidence and preparedness.

  • Mock Exams: Realistic practice tests designed to mirror the actual learners license examination.
  • Progress Tracking: Keep tabs on your performance and improvement over time.
  • Interactive Experience: Seamlessly integrated with the latest web technologies for an engaging user experience.
    • Our simulator is built using:
  • Django: A robust backend framework powering the core functionality.
  • HTMX: For smooth and interactive page updates without full reloads.
  • Alpine.js: Lightweight JavaScript to handle dynamic behaviours and responsive design.
  • TailwindCSS: An expressive open-Source CSS framework.

    • We are dedicated to making learning accessible and effective for everyone. Whether you're a first-time test taker or brushing up on your knowledge, our simulator is here to support you every step of the way. Feel free to contribute to our project and help us improve! Together, we can make the road to safe driving easier for all.
